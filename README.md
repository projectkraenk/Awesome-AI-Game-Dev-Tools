## AI-Native Engines & Builders
 
Full platforms that let you create playable games from text prompts or with AI-assisted workflows, often requiring little or no code.
 
- [Rosebud AI](https://rosebud.ai) — Prompt-to-game platform; describe an idea and get playable Three.js code, assets, and animations in the browser.
- [Summer Engine](https://www.summerengine.com) — AI-native engine built on Godot 4; free, no watermark/revenue share, exports to Steam/desktop. Includes a CLI and MCP server for 44+ engine actions.
- [Ztronica](https://ztronica.com) — AI-powered game creation and interactive experience builder for developers and creators.
- [GDevelop](https://gdevelop.io) — Free, open-source no-code 2D/3D engine with AI-assisted coding features and a large asset store.
- [Ludo.ai](https://ludo.ai) — AI game asset suite (sprites, UI, textures, music, 3D) plus market research and a Playable Generator for rapid prototyping.
- [Upit](https://upit.com) — AI suggests code and graphics from text prompts while still allowing manual editing; includes creator contests.
- [Gameer](https://www.gameer.io) — Play and create AI-generated interactive story games directly in the browser.
- [SeaVerse](https://seaverse.ai) — Make and publish playable 2D and 3D web games in minutes with AI; one-click publish and share.
- [Astrocade](https://www.astrocade.com) — Describe a game idea and AI builds a playable browser game instantly.
- [Star AI Game Generator](https://buildwithstar.com) — Free, generates a playable browser game from a prompt in ~30 seconds; no coding required.
- [Jabali.ai](https://www.jabali.ai) — Multiplayer-first AI game jams; includes modes like trivia, racing, and side-scrollers.
- [Promethean AI](https://www.prometheanai.com) — Populates full 3D environments from natural-language instructions; integrates with Unity and Unreal; used by PlayStation Studios.

## Code Assistants & Copilots
 
AI pair programmers and editors for C#, C++, GDScript, and other game-dev languages.
 
- [GitHub Copilot](https://github.com/features/copilot) — The most widely used AI code completion tool; works inside VS Code, JetBrains Rider, and Visual Studio.
- [Cursor](https://cursor.sh) — AI-first code editor with deep codebase understanding and an agent mode for multi-file edits.
- [Windsurf (Codeium)](https://codeium.com) — Free-tier AI coding assistant supporting 70+ languages including GDScript.
- [Claude Code](https://claude.ai) — Agentic CLI/IDE coding tool by Anthropic; commonly used for Unity, Godot, and Unreal scripting.
- [Aider](https://aider.chat) — Open-source AI pair programmer that edits code in your local git repo via terminal.
- [Continue.dev](https://continue.dev) — Open-source AI code assistant that plugs into VS Code and JetBrains; works with any model.

## 3D Asset Generation
 
Tools that turn text prompts or images into export-ready 3D models.
 
- [Meshy](https://www.meshy.ai) — Text/image-to-3D with PBR texturing, auto-rigging, 500+ animation presets, and Unity/Unreal/Blender plugins.
- [Tripo AI](https://www.tripo3d.ai) — Fast (~8–10 second) text/image-to-3D with auto-rigging and clean game-ready topology; GLB/FBX/OBJ export.
- [Rodin (Hyper3D)](https://hyper3d.ai) — High-precision 3D character and asset generation; strong on photorealism.
- [Alpha3D](https://www.alpha3d.io) — Full AI 3D pipeline: generate, refine topology/UVs/textures, rig, and export GLB, FBX, OBJ, STL.
- [Luma AI (Genie)](https://lumalabs.ai) — NeRF-based capture plus text-to-3D generation; strong for photorealistic environments.
- [Sloyd AI](https://www.sloyd.ai) — Parametric, slider-controlled 3D asset generation (adjust proportions precisely) rather than pure prompting.
- [3D AI Studio](https://www.3daistudio.com) — Aggregator giving access to multiple generation models under one dashboard.
- [Spline AI](https://spline.design) — Web-based, browser-collaborative 3D design with AI generation and real-time multiplayer editing.
- [Hunyuan3D (Tencent)](https://github.com/Tencent/Hunyuan3D-2) — Open-weight 3D generation model; widely used open-source option for organic shapes and characters.
- [TRELLIS (Microsoft Research)](https://github.com/microsoft/TRELLIS) — MIT-licensed open-source image-to-3D model.

## 2D Art, Sprites & Pixel Art
 
Generators for game-ready 2D assets, character sprites, sprite sheets, and pixel art.
 
- [PixelLab](https://www.pixellab.ai) — Pixel-art-specific generator with skeleton-based animation, text-to-animation, tilesets, and engine export.
- [Sprite-AI](https://www.sprite-ai.art) — Sprite generator, editor, and animator with MCP integration for Claude Code and Cursor.
- [AutoSprite](https://www.autosprite.io) — Upload a single sprite, pick a moveset, and export engine-ready sprite sheets for Unity, Godot, and GameMaker.
- [Sorceress](https://sorceress.games) — All-in-one suite: generate a character, animate with AI video, then auto-slice into a game-ready sprite sheet.
- [Scenario](https://www.scenario.com) — Custom-trained, style-consistent generation used by Ubisoft and InnoGames; Unity/Unreal plugins.
- [Leonardo AI](https://leonardo.ai) — General image generator with game-asset fine-tuned models and custom LoRA training.
- [Recraft](https://www.recraft.ai) — Strong for vector graphics (SVG) and consistent icon/UI sets.
- [Midjourney](https://www.midjourney.com) — High-end concept art and mood-board generation.
- [Stable Diffusion (ComfyUI)](https://github.com/comfyanonymous/ComfyUI) — Open-source local image generation with full control via node-based workflow.

## Texture & Material Generation
 
Tools for generating PBR texture maps and seamless tileable materials.
 
- [ArmorLab](https://armorpaint.org/armorlab) — Open-source, local PBR texture extraction from photos.
- [DreamTextures](https://github.com/carson-katri/dream-textures) — Open-source Blender plugin for generating seamless textures from text prompts via Stable Diffusion.
- [Polycam AI Texture](https://poly.cam) — AI texture generator that can wrap generated textures onto scanned 3D objects.
- [Adobe Substance 3D Sampler](https://www.adobe.com/products/substance3d-sampler.html) — Industry-standard photo-to-PBR material generation (Normal, Roughness, AO, etc.).

## Animation & Motion Capture
 
AI tools for creating, converting, and refining character animations.
 
- [Cascadeur](https://cascadeur.com) — AI- and physics-assisted keyframe animation (AutoPosing/AutoPhysics); free for non-commercial use.
- [DeepMotion](https://www.deepmotion.com) — Webcam/smartphone video to full-body 3D motion capture including face and hand tracking.
- [Move.ai](https://www.move.ai) — Markerless motion capture from standard camera video; multi-camera support for higher accuracy.
- [Rokoko Vision](https://www.rokoko.com/products/vision) — AI markerless mocap from ordinary video plus a smartphone-based facial capture app.
- [Krikey AI](https://www.krikey.ai) — Generative 3D avatar animation from prompts or reference video with voice-driven performance.
- [AnimationGPT](https://github.com/nixiCodex/AnimationGPT) — Open-source text-to-BVH motion generation trained specifically on game combat/action moves.
- [Plask](https://plask.ai) — Browser-based AI motion capture and animation editing for indie pipelines.
- [Mixamo (Adobe)](https://www.mixamo.com) — Free auto-rigging and large pre-made animation library; commonly paired with AI-generated meshes.

## NPC Dialogue, Behavior & AI Characters
 
Frameworks and platforms for building intelligent, conversational game characters.
 
- [Inworld AI](https://www.inworld.ai) — Industry-standard NPC platform with personality, memory, emotional state, and goal systems; Unity/Unreal plugins.
- [Convai](https://convai.com) — Real-time conversational NPCs with lip-sync, environmental awareness, and MetaHuman support.
- [NVIDIA ACE](https://developer.nvidia.com/ace) — Avatar Cloud Engine: speech, intelligence, and animation microservices for autonomous NPCs.
- [Eastworld](https://github.com/mluogh/eastworld) — Open-source (Apache 2.0) framework for NPCs with backstories, lore knowledge, and built-in world actions.
- [Gigax](https://github.com/GigaxGames/gigax) — Open fine-tuned LLMs for NPC interactions via function calling.
- [Unity ML-Agents](https://github.com/Unity-Technologies/ml-agents) — Open-source toolkit for training NPCs and agents with reinforcement learning inside Unity.
- [Beehave](https://github.com/bitbrain/beehave) — Open-source (MIT) behavior-tree AI addon for Godot.
- [behaviac (Tencent)](https://github.com/Tencent/behaviac) — Game AI development framework supporting Behavior Trees, FSM, and HTN; Unity/Unreal compatible.

## Voice, Speech & Dubbing
 
AI voice acting, text-to-speech, and in-game speech recognition tools.
 
- [ElevenLabs](https://elevenlabs.io) — Industry-leading TTS, voice cloning, multilingual dubbing, and sound-effect generation; dedicated gaming integrations.
- [Replica Studios](https://replicastudios.com) — AI voice actors with SAG-AFTRA licensing for games; emotion control.
- [Bark (Suno)](https://github.com/suno-ai/bark) — Open-source text-to-audio model that generates realistic speech, music, and ambient sound effects.
- [GPT-SoVITS](https://github.com/RVC-Boss/GPT-SoVITS) — Open-source few-shot voice cloning from 5 seconds of audio; cross-lingual support.
- [CosyVoice](https://github.com/FunAudioLLM/CosyVoice) — Multi-lingual large voice generation model; inference, training, and deployment.
- [Whisper (OpenAI)](https://github.com/openai/whisper) — Open-source speech-to-text for in-game voice commands or dialogue capture.

## Music Generation
 
AI tools for creating adaptive background music and full compositions for games.
 
- [Suno](https://suno.com) — Full songs with vocals and instrumentation from a text prompt; broad genre coverage.
- [Udio](https://www.udio.com) — High-fidelity music generation with stem export; strong for organic/live-instrument genres.
- [AIVA](https://www.aiva.ai) — Orchestral/cinematic composition with MIDI export; the most game-and-film-score-oriented option.
- [Soundraw](https://soundraw.io) — Mood/genre/tempo parameter music generator with stem export and structure editing.
- [Mubert](https://mubert.com) — Real-time, API-driven generative music for adaptive in-game soundtracks.
- [Beatoven.ai](https://www.beatoven.ai) — Built for game-dev workflows with scene-based mood control; Fairly Trained certified.
- [MusicGen (Meta)](https://github.com/facebookresearch/audiocraft) — Open-source text-to-music model.
- [ACE-Step](https://github.com/ace-step/ACE-Step-1.5) — Open-source music foundation model; generates full songs on consumer hardware.

## Sound Effects & Audio
 
Tools for generating and managing game sound effects and adaptive audio.
 
- [ElevenLabs Sound Effects](https://elevenlabs.io/sound-effects) — Text-to-SFX generation bundled with ElevenLabs subscriptions; royalty-free.
- [OptimizerAI](https://optimizerai.xyz) — Unlimited AI sound effect generation for games, videos, and apps.
- [SFXR / Bfxr](https://www.bfxr.net) — Classic free, open-source retro/chiptune sound-effect generator.
- [jsfxr](https://sfxr.me) — Browser-based JavaScript port of SFXR; no install required.
- [FMOD](https://www.fmod.com) — Industry-standard adaptive audio middleware; free under revenue threshold.
- [Wwise (Audiokinetic)](https://www.audiokinetic.com) — Industry-standard adaptive audio middleware; free under budget threshold.
- [Steam Audio](https://valvesoftware.github.io/steam-audio) — Free, open-source spatial audio and occlusion plugin by Valve for Unity, Unreal, FMOD, and Wwise.

## Level & Dungeon Generation
 
Tools for procedurally generating game levels, dungeons, and maps.
 
- [DungeonGenerator (UE5 Plugin)](https://github.com/happy-game-dev/DungeonGenerator) — Open-source Unreal Engine 5 plugin for generating grid-based 3D dungeons from parameters.
- [levelgen.ai](https://levelgen.ai) — Free procedural 2D level generation with theme-based visualization and export.
- [donjon RPG Tools](https://donjon.bin.sh) — Free random generators for dungeons, traps, NPCs, and world maps.

## World Models & Generative Engines
 
Research and production platforms that generate interactive game worlds end-to-end.
 
- [GameNGen](https://gamengen.github.io) — Diffusion models used as a real-time game engine (Google Research).
- [Genie 3 (Google DeepMind)](https://deepmind.google/models/genie-3) — General-purpose world model that generates photorealistic explorable environments from text.
- [Oasis (Decart)](https://oasis.decart.ai) — Interactive world model built on diffusion transformers; takes keyboard input and generates gameplay.
- [Rosebud AI Lab](https://lab.rosebud.ai) — Free online tool; describe a game and AI generates initial code, physics, and assets.

## Engine Integrations & MCP Tools
 
Plugins and bridges connecting AI models directly to Unity, Unreal, Godot, and Blender.
 
- [AnkleBreaker Studio — Unity MCP Server](https://github.com/AnkleBreaker-Studio/unity-mcp-server) — 288 AI-assisted tools for Claude/Cursor to control Unity Editor, scenes, builds, Shader Graph, NavMesh, and more.
- [AnkleBreaker Studio — Unity MCP Plugin](https://github.com/AnkleBreaker-Studio/unity-mcp-plugin) — Editor-side UPM package; pairs with the MCP Server above.
- [Godot MCP](https://github.com/Coding-Solo/godot-mcp) — MCP server that lets AI agents launch Godot, run projects, and capture debug output for autonomous correction.
- [BlenderMCP](https://github.com/ahujasid/blender-mcp) — Connects Blender to Claude AI via MCP; enables prompt-assisted 3D modeling and scene creation.
- [Unity AI](https://unity.com/features/ai) — Official Unity in-editor assistant, AI Gateway, and MCP Server for asset generation and agentic coding (Unity 6+).
- [ChatGPT-API-unity](https://github.com/hexthedev/ChatGPT-API-unity) — ChatGPT API binding for Unity C# (pure C#).
- [Hugging Face API Unity Integration](https://github.com/huggingface/unity-api) — Connects Unity projects to Hugging Face Inference API models.
- [UnityNeuroSpeech](https://github.com/unityneurospeech) — Unity framework for real-time voice AI: Whisper (STT) + Ollama (LLM) + XTTS (TTS), fully local.
